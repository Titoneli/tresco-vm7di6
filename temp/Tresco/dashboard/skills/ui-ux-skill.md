# BackDash: Resumo do Projeto para UI/UX

## 1. Conceito Central
**BackDash**: Resumo do Projeto para UI/UX é uma plataforma de backoffice que utiliza Inteligência Artificial para automatizar todo o processo de de análise de  fluxos e processos.

---

## 2. Identidade Visual e Estilo

*   **Cores Principais:**
    *   **Teal:** Confiança, cor primária da interface do painel.
    *   **Teal:** Destaques, botões de ação (CTAs), alertas.
    *   **Cinza Neutro (#F5F5F5):** Fundos e áreas de respiro.
*   **Tipografia:**
    *   **Cabeçalhos:** `Inter` (legível, profissional).
    *   **Corpo de Texto:** `Inter` (legível, profissional).
*   **Estilo Geral:**
    *   **Interface:** Limpa (clean), moderna e profissional. Escala de espaçamento 3px no CSS. Padronizar espaçamentos em múltiplos de 3 (design system) 
    *   **Componentes:** Baseado no **ShadCN/UI** (cantos arredondados, sombras sutis).
    *   **Ícones:** `Lucide-React` (ícones de linha, consistentes).
    *   **Responsividade:** O fluxo do cliente é **mobile-first**; o painel do consultor é otimizado para **desktop**.
